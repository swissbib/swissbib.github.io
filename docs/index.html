using Neural Networks Besides the availability of data, an additional reason why Machine Learning has become a popular topic has to do with publicly available code libraries that simplify the implementation of machines. One such library is TensorFlow, an open-source platform written by Google Brain. TensorFlow can be used for implementing Neural Networks. Since the end of 2019 a library called keras has been integrated on top of TensorFlow. This library makes the code implementation of Neural Networks even easier.
containers for libraries Every year, the apprentices of the University Library of Basel get the opportunity to get to know the tasks and projects of the IT department. A series of smaller workshops on selected topics are organized as part of these familiarization weeks. One of these workshops introduced the topic of container technologies. The author of the workshop took the topic as an opportunity to implement the swissbib discovery service with its various components (web application, search engine, database) as services in containers.
using Support Vector Machine Machine Learning may look mystical, although it is based on the mathematical discipline of statistics. If the learning process is done with a fixed data set, Machine Learning can be structured into two distinct groups of tasks, supervised and unsupervised learning. In supervised learning, the artificial machine is trained with a set of training data with a target value, each. In the first blog article, the feature records and their associated target value have been introduced for the swissbib project at hand.
using Ensemble Methods The notion of Machine Learning includes a wide group of statistical algorithms where a computer system learns on a set of training data and, after having completed its learning phase, uses its experience to generate predictions on new, unknown data. In the context of the capstone project of an advanced online training course at EPF Lausanne, a swissbib admirer takes the challenge to do Machine Learning with a set of library catalogue data in MARC 21 format.
